# 教师后台使用指南

> 版本：v2026.06 | 面向教师角色的后台操作说明

---

## 一、登录系统

1. 打开系统网址，进入登录页面。
2. 输入由管理员分配的账号和密码（角色为"教师"）。
3. 登录后自动进入教师监管后台。

![教师登录页](placeholder-login.png)

**注意**：如果登录后看到"无权限"提示，请确认账号角色是否为"教师"（联系管理员检查）。

---

## 二、全局概览

登录后默认展示**全局概览**页面，包含以下统计卡片：

| 卡片       | 说明                           |
|-----------|-------------------------------|
| 活跃房间数  | 当前正在进行的多人查寝房间数量       |
| 学生总数   | 系统中存储在寝状态的学生总数        |
| 各状态分布  | 在寝/离校/事假/外出/未归的人数       |

数据每 60 秒自动刷新，无需手动操作。

![全局概览](placeholder-overview.png)

---

## 三、房间监控

点击顶部"房间监控"Tab，可查看所有查寝房间列表：

- **房间卡片**：显示房间码（可复制）、创建者、过期倒计时、成员数量
- **活跃状态**：绿色圆点表示房间活跃，灰色表示已过期
- **点击卡片**：展开房间详情，查看：
  - 成员列表（姓名已脱敏）
  - 各学生当前状态及修改人
  - 房间操作日志

房间列表每 30 秒自动刷新。

![房间监控](placeholder-rooms.png)

---

## 四、报表导出

1. 点击顶部"报表导出"Tab。
2. 选择筛选条件：
   - **年级**：全部年级 / 指定年级
   - **楼层**：全部楼层 / 1F / 4F / 5F / 6F
3. 点击"导出 Excel"按钮。
4. 浏览器自动下载 `.xlsx` 文件，可用 Excel / WPS / LibreOffice 打开。

**注意**：教师导出的报表中，学生姓名自动脱敏（如"张**"）。

![报表导出](placeholder-export.png)

---

## 五、操作日志

点击顶部"操作日志"Tab，可查看系统操作记录：

- **筛选条件**：
  - 操作类型：数据上传 / 房间创建 / 状态变更 / 数据导出等
  - 操作人：输入用户名模糊搜索
  - 时间范围：最近 7 天 / 30 天 / 90 天
- **日志内容**：时间、操作人、角色、操作类型、详情、IP 地址
- **分页浏览**：底部翻页按钮，每页 20 条

**注意**：教师仅可查看房间相关日志和状态变更日志，无法查看管理操作（如数据上传、角色修改等）。

![操作日志](placeholder-logs.png)

## 六、常见问题

| 问题 | 解决方法 |
|------|---------|
| 看不到数据？ | 确认账号角色为"教师"，联系管理员检查权限设置 |
| 导出失败？ | 检查网络连接，或尝试缩小筛选范围（数据量过大时可能超时） |
| 房间列表为空？ | 当前无活跃查寝房间，等待查寝员创建房间后自动显示 |
| 登录过期？ | Token 有效期 30 天，过期后需重新登录 |
| 页面显示异常？ | 清除浏览器缓存后刷新，或尝试使用 Chrome/Safari 浏览器 |
